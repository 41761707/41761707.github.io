Trylma

GitHub

Dziennik

Aplikacja w języku Java imitująca funkcjonowanie dzienników szkolnych. Umożliwia podstawowe operacje związane z ocenianiem pracy uczniów. Do przechowywania danych wykorzystano relacyjną bazę MariaDB.

GitHub

Układanka

Edytor graficzny

GitHub

Kompilator

GitHub

Aplikacja bankowa
